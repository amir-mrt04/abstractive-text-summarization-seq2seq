# abstractive-text-summarization-seq2seq
Abstractive Text Summarization with Seq2Seq Models. Deep Learning final project (Spring 1404, Dr. Mir-Roshandel). Builds encoder-decoder (w/ attention) for news summaries on CNN/DailyMail (~300k articles). Two impls: Custom LSTM (PyTorch) &amp; fine-tuned BART (Transformers). ROUGE: R1~0.45, R2~0.22. Incl. notebooks, results, PDF report.
